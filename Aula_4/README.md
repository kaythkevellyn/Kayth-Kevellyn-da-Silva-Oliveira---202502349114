# Dashboard de Funcionários

Aplicação interativa desenvolvida com **Streamlit** para análise de dados de funcionários.
Permite visualizar métricas, filtrar informações e explorar os dados de forma simples e intuitiva.

## Funcionalidades

*  Filtro por cidade
*  Filtro por faixa salarial
*  KPIs automáticos (média, total e máximo)
*  Gráficos interativos
*  Tabela de dados dinâmica
*  Tabela dinâmica (pivot)
*  Download dos dados filtrados em CSV
*  Upload de arquivos CSV

---

## Tecnologias utilizadas

* Python
* Streamlit
* Pandas
* NumPy

## Como executar o projeto

### 1. Clonar o repositório

```
git clone https://github.com/seu-usuario/seu-repositorio.git
```

### 2. Acessar a pasta

```
cd seu-repositorio
```

### 3. Criar ambiente virtual

```
python -m venv .venv
```

### 4. Ativar o ambiente

**Windows:**

```
.venv\Scripts\activate
```

### 5. Instalar dependências

```
pip install -r requirements.txt
```

### 6. Rodar o projeto

```
streamlit run app.py
```
